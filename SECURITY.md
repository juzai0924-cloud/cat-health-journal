# Security

## 数据边界

本应用是无后端、本地优先的静态 Web App。猫咪资料、健康事件和照片只写入当前浏览器的 `localStorage`，不会上传 GitHub 或其他服务器。

公开仓库和在线页面仅包含应用代码与虚构 Demo 数据，不包含用户的本地记录。

## 输入与内容安全

- 所有导入对象均通过字段白名单和数量、长度上限校验后重建
- 图片仅接受 JPEG、PNG、WebP，拒绝 SVG、外部 URL 和超大文件
- 用户文本输出前进行 HTML 上下文编码
- CSP 仅允许同源脚本、样式、manifest、worker，以及受限的 data/blob 图片
- 不加载第三方脚本、统计 SDK 或广告 SDK
- Demo 数据仅存在于页面内存，刷新后自动重置

## 浏览器存储边界

`localStorage` 按浏览器 origin 隔离。正式在线使用版应部署在只承载本项目的独立 origin，避免与其他 GitHub Project Pages 共享 `https://juzai0924-cloud.github.io` 的存储边界。

本地数据未设置应用级密码；能够解锁设备并访问浏览器开发工具的人可能读取这些记录。该取舍与猫咪日常健康记录的敏感度相匹配，换取更低的使用门槛。

## 备份提示

导出的 JSON 备份包含完整文字记录和照片。请保存在可信设备中，不要公开上传或提交到 GitHub。清理浏览器数据或更换设备前应先导出备份。

## 报告问题

请勿在公开 Issue 中粘贴真实备份文件、猫咪健康记录或照片。
